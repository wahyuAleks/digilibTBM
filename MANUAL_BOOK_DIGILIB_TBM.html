üìñ Pendahuluan

Tentang digilibTBM

digilibTBM (Digital Library Taman Bacaan Masyarakat) adalah sebuah sistem manajemen perpustakaan digital yang dirancang untuk memudahkan pengelolaan koleksi buku, anggota, dan transaksi peminjaman buku secara online.

Fitur Utama

  • ‚úÖ Manajemen Buku: Tambah, edit, hapus, dan cari buku
  • ‚úÖ Katalog Digital: Pencarian buku berdasarkan judul, penulis, dan ISBN
  • ‚úÖ Peminjaman Online: Anggota dapat meminjam buku secara mandiri
  • ‚úÖ Pengembalian Mandiri: Anggota dapat mengembalikan buku secara online
  • ‚úÖ Sistem Denda: Perhitungan denda otomatis untuk keterlambatan
  • ‚úÖ Multi-User: Sistem role-based access (Admin & Member)
  • ‚úÖ Dashboard Interaktif: Visualisasi data dan statistik real-time

Role Pengguna

Role Akses Fungsi Utama
Admin Penuh Kelola buku, anggota, peminjaman, kategori, rak
Member/Anggota Terbatas Browse katalog, pinjam buku, kembalikan buku, lihat riwayat

üõ†Ô∏è Instalasi & Setup

Persyaratan Sistem

  • PHP 7.4 atau lebih tinggi
  • MySQL/MariaDB
  • Composer (untuk dependency management)
  • Web server (Apache/Nginx) atau PHP Built-in Server

Langkah-Langkah Instalasi

1. Clone/Download Project

cd d:\PROJEKTBM\digilibTBM

2. Install Dependencies

composer install

3. Setup Database

Opsi A: Menggunakan phpMyAdmin

  1. Buka phpMyAdmin
  2. Buat database baru bernama db_digilib_tbm
  3. Import file SQL: SQL_UNTUK_PHPMYADMIN.sql

Opsi B: Menggunakan Script Otomatis

.\RUN_CREATE_DATABASE.bat

4. Jalankan Server

.\START_SERVER.bat

atau

php yii serve --port=8080

5. Akses Aplikasi

Buka browser: http://localhost:8080

üéØ Fitur-Fitur Sistem

Struktur Database

Sistem menggunakan 8 tabel utama:

  • USER - Data pengguna sistem
  • ANGGOTA - Data anggota perpustakaan
  • BUKU - Koleksi buku
  • KATEGORI - Kategori buku
  • RAK - Lokasi penyimpanan buku
  • PEMINJAMAN - Transaksi peminjaman
  • ITEM_PEMINJAMAN - Detail buku yang dipinjam
  • DENDA - Denda keterlambatan

Kategori Buku Default

  1. Fiksi - Novel, cerita pendek, dll
  2. Non-Fiksi - Biografi, sejarah, dll
  3. Pendidikan - Buku pelajaran, tutorial, dll
  4. Teknologi - Programming, IT, dll

Status Peminjaman

Status Deskripsi
menunggu_verifikasi_admin Peminjaman baru menunggu approval admin
dipinjam Buku sedang dipinjam (sudah diverifikasi)
dikembalikan Buku sudah dikembalikan

Meminjam Buku Online

Cara Pinjam Buku

  1. Browse katalog dan temukan buku yang diinginkan
  2. Klik "Pinjam Sekarang"
  3. Sistem membuat record peminjaman dengan status menunggu_verifikasi_admin
  4. Ambil buku di perpustakaan setelah mendapat konfirmasi admin
üìå Catatan: Peminjaman menunggu verifikasi admin. Ambil buku di perpustakaan setelah mendapat konfirmasi.

üîß Troubleshooting

Error: "Connection Refused"

Penyebab: Server tidak berjalan

Solusi:

.\START_SERVER.bat
atau
php yii serve --port=8080

Error: "Database Connection Failed"

Penyebab: Konfigurasi database salah atau MySQL tidak berjalan

Solusi:

  1. Cek service MySQL/XAMPP sudah running
  2. Verifikasi kredensial di config/db.php
  3. Pastikan database db_digilib_tbm sudah dibuat

Error: "Duplicate Entry for key 'email'"

Solusi: Gunakan email yang berbeda atau hapus user lama dari database

Buku Tidak Bisa Dipinjam

Kemungkinan penyebab:

  • Stok habis (0)
  • Sudah ada peminjaman aktif yang belum dikembalikan
  • Akun belum diverifikasi admin

Lupa Password

Untuk Anggota: Hubungi admin perpustakaan

Untuk Admin: Update password via phpMyAdmin:

UPDATE user 
SET passwordHash = '12345' 
WHERE email = 'admin@digilib.com';

‚ùì FAQ (Frequently Asked Questions)

1. Berapa lama durasi peminjaman buku?

Jawab: Default durasi peminjaman adalah 14 hari kalender.

2. Apakah bisa memperpanjang peminjaman?

Jawab: Saat ini sistem belum support perpanjangan otomatis. Hubungi admin perpustakaan.

3. Bagaimana cara membayar denda?

Jawab: Denda dibayarkan tunai di perpustakaan saat mengembalikan buku fisik.

4. Apakah bisa meminjam lebih dari 1 buku sekaligus?

Jawab: Ya. Pinjam buku satu per satu melalui katalog. Semua peminjaman tercatat terpisah.

5. Kenapa peminjaman saya masih "Menunggu Verifikasi"?

Jawab: Peminjaman online perlu verifikasi admin. Datang ke perpustakaan untuk mengambil buku.

6. Bagaimana cara backup database?

Jawab via phpMyAdmin:

  1. Buka phpMyAdmin
  2. Pilih database db_digilib_tbm
  3. Klik tab "Export"
  4. Pilih format SQL
  5. Klik "Go"

Via Command Line:

mysqldump -u root -p db_digilib_tbm > backup_digilib.sql

7. Bagaimana cara restore database?

Jawab via phpMyAdmin:

  1. Buat database baru db_digilib_tbm
  2. Klik tab "Import"
  3. Pilih file backup SQL
  4. Klik "Go"

8. Bagaimana cara mengubah tarif denda?

Jawab: Edit file controllers/AnggotaController.php di method actionPengembalianMandiri:

$tarifDendaPerHari = 1000; // Ubah nilai ini
